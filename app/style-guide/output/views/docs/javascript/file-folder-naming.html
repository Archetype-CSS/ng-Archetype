A3 File/Folder Structure & Naming Conventions

Tl;Dr

This document seeks to answer the simple question, "Where the F@#K is that file?!". Our approach is influenced by the "Modularity" example in this article. For more information on modules, see the Angular Style Guide.

Folder Structure

Files are generally organized one of two ways.

  • In a module folder that is specific to an application function (residents, MDS, users, etc), or...
  • In a shared folder if it's a reusable resource (components, images, directives, services etc)
app/
|__assets/
   |__common/
      |__fonts/
      |__images/
      |__javascripts/
         |__directives/
         |__filters/
         |__services/
         |__...
   |__components/

      |__...

         |__...
      |__users/

      |__...
   |__assess/

   |__improve/
   |__...
